Retargeting Lab

Isolated retargeting experiments (not used by runtime avatars).

Why this exists

Mixamo reuse is a skeleton-compatibility and deformation-quality problem. This lab is where we validate mapping, rest-pose alignment, translation policy, IK, and deformation fixes before touching production code.

Lab assets
  • PLAYBOOK.md - strategy + pipeline recommendations
  • EXPERIMENT-MATRIX.md - test cases and pass/fail criteria
  • PROGRESS-SUMMARY.md - what has been implemented and observed so far
  • bone-map.mixamo-core.json - starter mapping template
  • generate-meshy-refine-batch.ts - 5-sample refine-only batch + report
  • rig-sweep.config.json - autorig variable variants
  • run-rig-sweep.ts - config-driven autorig sweep + compatibility report
  • publish-rig-sweep-to-test-env.ts - publish latest sweep models into test viewer
  • test-rig.html - current visual sandbox
Rule

No retargeting logic gets added back into client/js/avatars.js until it is stable and repeatable in this lab.
